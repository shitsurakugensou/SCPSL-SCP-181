## What is that | 这是什么鬼
This is a server plugin for SCP: SL.
这个是一个给SCP SL服务器做的插件

## Features | 功能
- Randomly chose a D-Class personnel become the SCP-181.
- SCP-181 can get some randomly chance to open the door that KeyCard needed.
- SCP-181 can get some randomly chance to immunology other SCP's attack.
- When SCP-181 escape, He will get a MicroH.I.D or E11, flash bang and a weapon-manager tablet dependent which team that he joined.
- 随机抽取一个DD当SCP-181
- SCP-181 可以随机访问权限门
- SCP-181 可以随机免除SCP攻击伤害
- SCP-181 可以获得 电磁炮(NTF) 或 E11步枪, 手雷和平板(Chaos)。


## Config Item | 配置项

### English
|      Key      |  Value Type  | Default Value | Description |
| :-----------: | :----------: | :-----------: | :---------: |
| scp181_enable |   Boolean    |     true      | Enable/Disable the plugin
| scp181_debug  |   Boolean    |     false     | Enable/Disable the debug mode
| scp181_prefix_display | Boolean | true       | Will the badge be display
| scp181_door_open_chance | int [1~100] | 6    | The percentage that SCP-181 will open the Door
| scp181_dodge_chance     | int [1~100] | 6    | The percentage that SCP-181 will get 0 damage

### 中文
|      项       |    数据类型   |      默认值    |     说明     |
| :-----------: | :----------: | :-----------: | :---------: |
| scp181_enable |   布尔值    |     true      | 启用/禁用 SCP-181
| scp181_debug  |   布尔值    |     false     | 启用/禁用 调试模式
| scp181_prefix_display | 布尔值 | true       | 是否显示头衔
| scp181_door_open_chance | 整数 [1~100] | 6    | SCP-181 开门的几率
| scp181_dodge_chance     | 整数 [1~100] | 6    | SCP-181 免伤的几率
